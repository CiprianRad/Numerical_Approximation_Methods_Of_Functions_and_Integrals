# Numerical Approximation Methods of Functions and Integrals

## 📌 Overview
This repository contains a collection of **MATLAB scripts** designed for numerical approximations of functions and integrals. The scripts serve as practical implementations of mathematical methods, showcasing real-life examples and utilizing interactive graphs to help visualize complex numerical behaviors.

## 📂 Repository Structure
The scripts are grouped into categorized folders based on the numerical method being applied:

* **`Intro/`** - Basic introductory scripts to set up the foundation for numerical approximations in MATLAB.
* **`Hermite_Interpolation/`** - Scripts implementing Hermite interpolation methods for estimating functions using both value and derivative data.
* **`Lagrange_Interpolation/`** - Examples demonstrating the Lagrange polynomial interpolation technique.
* **`Least_Squares/`** & **`Least_Squares./`** - Implementations of the method of least squares for curve fitting and regression analysis.
* **`Newton_Poly_And_Neville_Algorithm/`** - Scripts utilizing Newton's divided differences interpolation polynomial and Neville's algorithm for iterative interpolation.
* **`Orthogonal_Poly_And_Divided_Differences_Tables/`** - Computation and application of orthogonal polynomials and divided difference tables.
* **`Spline_Interpolation/`** - Demonstrations of piecewise polynomial interpolation using splines to ensure curve smoothness.

## 🚀 Getting Started

### Prerequisites
To run these scripts, you will need to have [MATLAB](https://www.mathworks.com/products/matlab.html) installed on your system.

### Usage
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/CiprianRad/Numerical_Approximation_Methods_Of_Functions_and_Integrals.git

2. Launch MATLAB and set your current working directory to the cloned repository.

3. Navigate into the specific folder of the method you wish to explore.

4. Open and run the .m scripts to view the computations, generated interactive graphs, and real-life examples.
